# Health_Together

헬스장 커뮤니티 어플리케이션
- 개발인원: 개인 프로젝트
- 개발환경: Android Studio
- 언어: Java
- 개발 배경:
• 다양한 운동 종목 가운데, 헬스만 커뮤니티 문화가 약해 헬스장에서 개인, 소규모의 운동 문
화가 이루어짐
• 사용자가 다니는 헬스장을 기준으로 커뮤니티를 조성해 다 같이 운동할 수 있는 문화 조성
을 위해 개발
- 기능:
• Volley를 이용한 http통신
• Google maps, places를 이용한 장소 찾기
• SharedPreferences를 이용한 사용자 관리
